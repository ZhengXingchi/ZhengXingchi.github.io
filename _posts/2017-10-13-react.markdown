## 配置环境
- 安装NodeJS以及安装yoeman`npm install -g yo`
查看yo版本`yo --version`

- 安装react-webpack`npm install -g generator-react-webpack`
查看react-webpack版本`npm ls -g --depth=1 2>/dev/null | grep generator-`
